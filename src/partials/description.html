Акумулюючі ємності WELDER для тепло- та холодопостачання

Наша буферна ємність виготовляються з низьковуглецевої сталі товщиною 3-4мм. Кожна ємність має антикорозійне покриття з зовні та всередині для створення водонепроникного бар'єру.

Для високої теплоізоляції буферної ємності утеплюється пінополіуретаном (німецького виробництва)товщиною 50 мм, який забезпечує хороший рівень термо-, гідро- та шумоізоляції.
